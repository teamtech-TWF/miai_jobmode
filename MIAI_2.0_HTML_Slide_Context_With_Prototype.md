# 2. Core Positioning

## One-Line Definition

**MiAI 2.0 is a governed AI strategy runtime that turns messy MI inputs into evidence-grounded, human-approved, client-ready artifacts.**

## Short Positioning Statement

MiAI 2.0 is not a chatbot. It is a human-in-the-loop strategy runtime for Media Intelligence and Strategy teams.

It helps transform:

```text
Messy Briefs / Files / Data
→ Structured Context
→ Evidence Retrieval
→ Specialist Agent Workflow
→ Human Review
→ Versioned Artifact
→ Final Output
```

## Why This Matters

MI already has strong strategic thinking, frameworks, and working processes. The challenge is that work often starts from messy, scattered, incomplete inputs.

MiAI 2.0 helps standardize how MI works with AI by adding:

- Structured context before generation
- Evidence grounding before recommendation
- JobMode-based workflow selection
- Specialist agents for different MI tasks
- Human review and approval checkpoints
- Versioned artifacts and regeneration
- Final export-ready outputs
