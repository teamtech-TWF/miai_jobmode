# MIAI 2.0 - Job Mode & Artifacts

Facilitation Guide and Input Capture Sheet for the TWF Working Session

Prepared for review by 3 AI super users. Output to be handed to TWF. Confidential.

## One-Sentence Goal

For each media-planning job, agree:

- the job mode
- the primary artifact it produces
- the artifact details TWF needs to build the agent

Where reviewers disagree, capture the disagreement instead of smoothing it over.

## Definitions

### Job

One discrete task a planner does, for example:

- build audience segments
- analyze marketing situation

### Job Mode

The type of run you launch. It determines which agents activate and which artifacts are produced.

Modes referenced in the source:

- Context Build
- Diagnosis
- Strategy Development
- Storytelling / Presentation
- Full End-to-End Run

### Primary Artifact

The main output the job or mode produces and hands to the next step.

## Job to Mode Reference

| # | Job Mode | What It Does | Primary Artifact | Format |
| --- | --- | --- | --- | --- |
| 1 | Context Builder | Structure messy input to trusted context; create live research | Context summary; context report | `.docx`, `.pdf` |
| 2 | Diagnosis | Analyze business issues, market and consumer trends, and opportunity gaps | Context summary; context report | `.docx`, `.pdf` |
| 3 | Strategy Development | Generate or recommend strategic direction, options, or ideas | Strategy blueprint | `.docx`, `.pdf` |
| 4 | Storytelling Presentation | Turn content and strategy into presentation narrative | Storyline; ready-to-use prompt for Beautiful.ai; presentation draft | `.pptx` |

## Facilitation Tip

Print one grid per reviewer so answers stay independent, then reconcile on a shared copy. If the three reviewers disagree on a job, preserve the disagreement. That is useful input for TWF.

## Confusion and Open Questions Log

Capture:

- the question or point of confusion
- who raised it

## Handoff to TWF

After the session, provide:

- the completed grid linking each job to mode, artifact, format, inputs, and agent extras
- a job-mode to artifacts summary extended across all five modes
- a consolidated confusion log keyed to deck slides
